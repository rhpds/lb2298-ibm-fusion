Welcome to the Modernizing Virtualization with Red Hat OpenShift Virtualization and IBM Fusion hands-on lab!

== What you will learn

In this workshop, you will:

* Explore the OpenShift Virtualization and IBM Fusion environment
* Create and manage virtual machines on Red Hat OpenShift
* Work with IBM Fusion-backed persistent storage for VM workloads
* Perform VM lifecycle operations including live migration, snapshots, and cloning
* Plan and execute workload migrations from legacy virtualization platforms

== Who this is for

This workshop is designed for architects, SREs, infrastructure engineers, and platform engineers who want to learn how to modernize virtualization workloads using Red Hat OpenShift Virtualization integrated with IBM Fusion.

**Experience level**: Beginner to Intermediate

== Prerequisites

Before starting this workshop, you should have:

* Basic familiarity with Red Hat OpenShift concepts (projects, pods, operators)
* General understanding of virtualization technology (VMs, hypervisors, storage)
* Comfort working with a web-based console and command line interface
